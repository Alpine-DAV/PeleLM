Getting Started

  • Clone

    git clone https://github.com/AMReX-Codes/amrex.git
git clone https://github.com/AMReX-Codes/IAMR.git
git clone https://github.com/AMReX-Combustion/PelePhysics.git
git clone https://github.com/AMReX-Combustion/PeleLM.git

  • Setup Your Environment

    export AMREX_HOME=/path/to/AMReX
export IAMR_HOME=/path/to/IAMR
export PELE_PHYSICS_HOME=/path/to/PelePhysics
export PELELM_HOME=/path/to/PeleLM

  • Compile

    problems are compiled in their own directory. E.g., 
    cd PeleLM/Exec/FlameInABox/
make DIM=2 COMP=gnu DEBUG=FALSE USE_MPI=TRUE

  • Run

    To run a simulation, an input file is always required: 
     mpiexec -n 2 ./PeleLM2d.gnu.MPI.ex inputs.2d-regt

  • Visualize

    AMReX output is supported natively by yt, VisIt, Amrvis, and ParaView.

The PeleLM User's Guide is currently in the PeleLM/UsersGuide directory.
The AMReX Documentation is the best place to learn more about how the AMReX source code is organized.
You can also ask for help by opening up an issue on the PeleLM GitHub webpage.
